This project is a decentralized advertising platform built using Soroban smart contracts on the Stellar blockchain. It enables advertisers to publish and manage advertisements in a transparent, censorship-resistant, and trustless environment.

Traditional advertising platforms are centralized, opaque, and often biased. This project aims to remove intermediaries and give full control back to advertisers and users.

---

## ⚙️ What it does

This smart contract allows:

- Advertisers to create and store ads on-chain  
- Retrieval of ads by ID  
- Viewing all published ads  

Each advertisement contains:
- Unique ID  
- Owner (wallet address)  
- Title  
- Content  

All data is stored on the blockchain, ensuring immutability and transparency.

---

## ✨ Features

- 🧾 On-chain ad storage  
- 🔐 Ownership tied to wallet addresses  
- 📦 Simple and extensible contract design  
- 🌐 Fully decentralized (no central authority)  
- ⚡ Built on Soroban (fast + low-cost execution)
